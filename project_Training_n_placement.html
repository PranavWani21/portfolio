Highlights of Project

1) After landing page, there is login page for authorized users (Admin/TNP Faculty/Registered students). The Faculty and student login page is same as Admin login.
2) After login into the web app, users will be having a Dashboard with some stats in blocks with there name on top and the sidebar on left. Whole website is dynamic so any changes in data will automatically gets reflected.
3) Student table with data of all students. If you want to see previous batches record just click on button provided above, it will opens a modal and you can see the students details of respective batch. You can add, edit/update, Search, the student record and can delete as well.
4) Creating user / registration is allowed only from inside of the app. You can search and delete the user.
5) Analysis of stored data. You can take the print of reports also.
6) Authenticity: If student got offer letter then he/she has to inform to admin/tnp faculty first and then only he/she is able to update placement related fields, else profile is not updated.
7) Security: Each module is isolated from others. for ex. Student could not access admin pages.
